Practical Task

Set up and optimize a YouTube channel

Create a properly configured channel with SEO-ready branding.
Step-by-Step Method
  1. Create a new YouTube channel with a clear niche-based name.
  2. Design a professional banner in Canva using 2560×1440 pixels.
  3. Write a keyword-rich channel description (150 to 200 words).
  4. Add your social links and website if applicable in the channel settings.
  5. Create a channel trailer script of 30 to 60 seconds introducing the channel.
DeliverableA fully set up YouTube channel ready for first video upload.
Pro TipYour channel name should clearly signal the niche — do not use a personal name for a niche channel.
Practical Task

Design 3 thumbnails for the same video topic

Learn thumbnail A/B testing principles and design for high CTR.
Step-by-Step Method
  1. Choose one video topic and open Canva with a 1280×720 template.
  2. Create version 1 — bold text, one face, high contrast background.
  3. Create version 2 — curiosity-based text, emotion-driven face, minimal background.
  4. Create version 3 — numbered list style with clean icon or graphic.
  5. Compare all 3 and choose the one you would most likely click as a viewer.
Deliverable3 thumbnail variations saved as PNG files, ready for upload.
Pro TipThe best thumbnail creates curiosity without revealing everything — leave something for the viewer to want.
Practical Task

Write a complete SEO-optimized video description

Learn to write descriptions that help videos rank in YouTube search.
Step-by-Step Method
  1. Research the primary keyword for your video using TubeBuddy or vidIQ.
  2. Write the first 2 sentences with the keyword placed naturally.
  3. Add 3 to 5 supporting keywords in the middle of the description.
  4. Include a call-to-action for subscribe, comment or playlist.
  5. Add chapter timestamps if the video is longer than 5 minutes.
DeliverableOne complete YouTube video description ready to copy and paste.
Pro TipYouTube reads only the first 2 lines before the 'Show More' — make them count.
