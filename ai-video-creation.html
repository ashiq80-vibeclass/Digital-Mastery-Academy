Practical Task

Generate a 30-second AI video from a text prompt

Learn the prompt-to-video workflow using Sora AI or VEO-3.
Step-by-Step Method
  1. Open Sora AI or VEO-3 and create an account if not already set up.
  2. Write a detailed text prompt describing the scene, mood, lighting and style.
  3. Generate the video and review the output for quality and relevance.
  4. If the output needs improvement, refine the prompt with more specific language.
  5. Download the best version and note what made the prompt work well.
DeliverableA 30-second AI-generated video clip with the prompt that created it saved for reference.
Pro TipSpecific prompts produce better videos. Describe the scene like a film director briefing a camera operator.
Practical Task

Create a faceless explainer video using Hedra AI

Build a complete AI avatar video with script and voiceover.
Step-by-Step Method
  1. Write a 60-second script on any educational topic.
  2. Use ElevenLabs or any text-to-speech tool to generate a clear voiceover.
  3. Upload the voiceover to Hedra AI and select or generate an AI avatar.
  4. Generate the avatar video and download the output.
  5. Import into CapCut Pro, add captions and a background music track.
DeliverableA 60-second faceless explainer video with AI avatar, captions and music.
Pro TipThe voiceover quality is more important than the avatar quality — invest time in the script and audio first.
Practical Task

Edit and deliver a complete AI promotional video

Practice the end-to-end workflow for a client-ready promotional video.
Step-by-Step Method
  1. Define the product or service the video is promoting.
  2. Generate relevant AI scenes using Sora or VEO-3 for different sections.
  3. Write a script and generate a voiceover for the narration.
  4. Assemble the video in CapCut Pro with scenes, voiceover, captions and brand colors.
  5. Export in 1080p and review the final video as if you were the client.
DeliverableA 60 to 90-second promotional video ready for client delivery.
Pro TipAlways watch your final video from start to finish before delivering. One mistake is all a client needs to lose trust.
