Build professional websites from scratch. Learn domain and hosting setup, WordPress installation, UI/UX design principles, and how to create responsive, launch-ready sites on Pantheon.

Course Overview

What You Will Learn

  • Understand domains, hosting, DNS settings and how websites go live on the internet.
  • Install WordPress and configure essential settings, plugins and themes.
  • Design professional page layouts with proper UI/UX principles.

Phase 2 — Intermediate

  • UI/UX principles for website layouts

  • Contact forms, image optimization and SEO settings

Phase 3 — Professional

  • Pantheon environment setup for professional hosting
  • Launching a live website and DNS configuration
  • Speed optimization and performance testing
  • Handing over a client-ready WordPress site

Interactive Practical Work

Click Any Task to See Full Method

Practical Task

Build a complete 3-page WordPress website

Practice full website creation with home, about and contact pages.
Step-by-Step Method
  1. Install WordPress on Pantheon and complete the basic settings setup.
  2. Install Elementor and choose a clean starter template.
  3. Build a homepage with a hero section, features section and CTA button.
  4. Build an about page with a brief description, image and team section.
  5. Build a contact page with a working contact form plugin.
DeliverableA 3-page WordPress website with consistent design and working navigation.
Pro TipUse no more than 2 fonts and 3 colors — simplicity makes professional sites stand out.
Practical Task

Optimize a WordPress website for speed

Learn basic performance optimization that improves user experience and SEO.
Step-by-Step Method
  1. Install a caching plugin such as WP Fastest Cache or W3 Total Cache.
  2. Install a free image optimization plugin and compress all uploaded images.
  3. Disable any unused plugins and remove default WordPress themes.
  4. Test the site on Google PageSpeed Insights before and after optimization.
  5. Screenshot both results to show the improvement.
DeliverableA PageSpeed report showing improvement score after optimization.
Pro TipA slow website loses visitors before they even read your content. Speed is a direct ranking factor.
Practical Task

Create and deliver a client project website

Practice the complete workflow from briefing to client handover.
Step-by-Step Method
  1. Define the website goal — portfolio, business, blog or landing page.
  2. Choose appropriate colors, fonts and layout based on the client niche.
  3. Build all required pages with consistent header and footer.
  4. Test the site on mobile, tablet and desktop screens.
  5. Export a list of login credentials and usage instructions for client handover.
DeliverableA complete website with login credentials and a handover checklist.
Pro TipClients remember who made the process easy. Clear handover documentation builds long-term trust.
Teaching Flow

Recommended Teaching Sequence

  • Walk students through the Pantheon account creation and environment setup.
  • Build a homepage section live in Elementor while students follow along.
  • Show the difference between a good and bad UI layout using real examples.
  • Test a student site on PageSpeed Insights together and explain the results.
  • Demonstrate the domain connection and SSL activation process.
