Practical Task

Write a 600-word SEO blog post

Practice keyword placement, headings structure and SEO-friendly formatting.
Step-by-Step Method
  1. Choose a simple topic and find one primary keyword using Google Keyword Planner.
  2. Write an engaging introduction that includes the keyword naturally in the first paragraph.
  3. Create three H2 subheadings that cover the main points of the topic.
  4. Write 2 to 3 paragraphs under each heading — clear, useful and readable.
  5. Write a short conclusion with a call-to-action at the end.
  6. Run the article through a plagiarism checker and fix any flagged sections.

Pro TipWrite for the reader first, then optimize for search engines second.
Practical Task

Write a YouTube script for a 5-minute video

Learn script structure, hooks and viewer retention techniques.
Step-by-Step Method
  1. Choose a topic your target audience would search on YouTube.
  2. Write a strong hook in the first 15 seconds — a question, stat or bold statement.
  3. Outline 3 main points that answer the viewer's search intent.
  4. Write conversational script text under each point — as if speaking to one person.
  5. End with a clear call-to-action — subscribe, comment or visit a link.
Deliverable1 complete YouTube script with hook, 3 sections and CTA.
Pro TipRead your script out loud. If it sounds unnatural, rewrite it in simpler language.
Practical Task

Rewrite an AI-generated paragraph to remove plagiarism

Learn to use AI assistance while maintaining originality.
Step-by-Step Method
  1. Take any AI-generated paragraph of 100 to 150 words.
  2. Read it carefully and identify phrases that sound generic or copied.
  3. Rewrite each sentence in your own words without changing the meaning.
  4. Change sentence structure, synonyms and paragraph order.
  5. Run the rewritten version through Copyscape or Duplichecker and confirm it passes.

Pro TipGood paraphrasing means restructuring the idea — not just swapping words.
Teaching Flow

Recommended Teaching Sequence

  • Start by showing strong and weak writing examples side by side.
  • Teach students to identify keywords before writing any piece of content.
  • Write one short blog introduction together as a class exercise.
  • Ask students to rewrite a sample paragraph and share their version.
  • Review grammar, structure and keyword use in student submissions.
